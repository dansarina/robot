Dansarina

Download: dansarina-0.63.tar.gz

Dansarina is a fast, portable, event-driven IRC robot. She depends very little on the libc, and compiles on any modern UNIX system. At start up, dansarina connects to the IRC server and listens on a port for a possible telnet connection, so that an administrator can control her without joining the IRC network. She has a plugin system that works like a shell --- possible security concerns lies in the plugins, not in dansarina; she does not use system() or popen(). IRC clients can connect to her through a DCC chat after authentication.

Up to version 0.60, she can link herself to another dansarina for a robot network. She comes with a script system --- she is able to execute scripts and reply back the script outputs, so users can look up words in dictionaries on the web and things like that. (You must write the script yourself, but it can be in any language. It's the UNIX system itself that runs them. Yes, security is your responsibility.)

Installation

Just unpack and say ``make.'' She'll run in the same directory. If you'd like to install somewhere else, just copy dansarina, plugger, and the directories run/, ctl/, and hlp/. That's all. After unpacking, her configuration is in ctl/. The files are intuitive. Always write one whole line in these files; that is, end your strings with \n; otherwise she'll complain.

About plugger

Plugger is the plugin daemon; dansarina executes it at start up and connects a pipe to it. They have a simple protocol of communication and plugger manages the whole overhead of the plugins. Plugins do not execute in sequential order; a slow one --- such as one that queries the network --- may begin first, and end later than another one; plugger takes care of running them in background.

See this for a detailed description of the implementation of plugger.
